# 🏅 SportifyGear - Project Setup

This project has been set up to run directly on your host machine.

### 🏁 Quick Start

1. **Install Dependencies**:
   ```bash
   composer install --ignore-platform-reqs
   npm install
   ```

2. **Migrate & Seed**:
   ```bash
   php artisan migrate:fresh --seed
   ```

3. **Run**:
   - `php artisan serve`
   - `npm run dev`

Check the main [README.md](../../README.md) in the root folder for the complete step-by-step guide! 🚀
